What is Intelligence Analysis?

Image from pixabay.com

Intelligence analysis is a support function that helps drive operations in a tactical environment, provide insight in a strategic environment, and uses a specialized set of tools and techniques in order to transform raw data and information into actionable intelligence. The important phrase here is, “actionable intelligence.”
